This repository contains the following files:

* README.md: this README file
* CodeBook.md: the code book describing the variables, the data, and any 
transformations or work performed to clean up the data
* run_analysis.R: the R script used to read, clean up, and write the data. 
It assumes that the data is in a "UCI HAR Dataset" folder in the working directory
* tidy.txt: the output file produced by run_analysis.R script
